# react-bind-html

Parse HTML string and display in react application

## Install

```
npm install react-bind-html
```

## Usage

```
/* import */
import BindHtml from 'react-bind-html';

/* Invocation */
<BindHtml html="<i>Italic styled content</i>" />
```

## Output

### Generated content

```
<i>Italic styled content</i>
```

### HTML Output

<i>Italic styled content</i>
